# PDF_Crawler.py
Find WORD and the Page_number of the WORD in a PDF file with Python

Uses Python 3.7

Usage:

linux:
#pip3 install regex
#pip3 install PyPDF2
#python3 PDF_Crawler.py

Windows:
#pip install regex
#pip install PyPFD2
#python PDF_Crawler.py

Put in the .pdf file install the folder of PDF_Crawler.py
